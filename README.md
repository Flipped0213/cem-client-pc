# cem-client-pc

## 电商管理平台

该项目是利用Vue3和Vue2以及相关的基于Vue3的组合API项目，该项目实现了完整电商所具有的完整得电商业务功能，采用前后端分离模式，独立完成各部分业务。

## Project setup
```
yarn install
```
